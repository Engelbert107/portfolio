This project explores various anomaly detection algorithms and employs rules like the 80% and Cattell's rules in Principal Component Analysis (PCA) to identify key components contributing to data variation. Wine Quality Classification using Machine Learning

In this project, we classified wine quality using the UCI Wine Quality dataset, focusing on predicting whether wines fall into lower or higher quality categories based on their chemical properties. After data preprocessing and normalization, we trained six models, including K-Nearest Neighbors, Random Forest, Gradient Boosting, LightGBM, and XGBoost.

The Random Forest model performed the best, achieving 81% accuracy and a ROC-AUC score of 0.897. Despite some false positives, it showed strong precision and recall across classes. We deployed the model locally using Flask, making it ready for practical applications in wine quality classification.
